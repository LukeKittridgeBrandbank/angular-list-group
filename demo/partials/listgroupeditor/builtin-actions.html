Use deletable attribute to add a delete action button

Use on-delete attribute to specify a callback function.

Note argument function name $item 

<list-group 
	title="Built-in delete action" 
	items="colors"
	deletable
	on-delete="deleteItem($item)"> 
</list-group>

Use editable attribute to add an edit action button

Use on-edit attribute to specify a callback function.

Note argument function name $item 

<list-group
	title="Built-in edit action"
	items="colors02"
	editable
	on-edit="editItem($item)"> 
</list-group>

Add both built-in actions edit and delete

<list-group
	title="Built-in edit and delete actions"
	items="colors03"
	deletable
	on-delete="deleteItem($item)"
	editable
	on-edit="editItem($item)"> 
</list-group>
