In its most basic usage the listGroup directive simply renders the specified items as a Boostrap list group component.

Controller

$scope.colors=[ 'Red', 'Orange', 'Yellow', 
	'Green', 'Blue', 'Indigo', 'Violet' ];

Markup

<list-group items="colors"></list-group>
In case items are objects, they are rendered applying angular.toJson() to each item

Controller

$scope.colors2 = [ {
	'name' : 'Red',
	'hex' : '#FF0000'
   }, {
	'name' : 'Orange',
	'hex' : '#FF7F00'
   } ];

Markup

<list-group items="colors2"></list-group>
If objects have a property named label then the value of this property will be used to render the items

Controller

$scope.colors3 = [ {
	'label' : 'Red',
	'hex' : '#FF0000'
   }, {
	'label' : 'Orange',
	'hex' : '#FF7F00'
   } ];

Markup

<list-group items="colors3"></list-group>
If none of this fits your needs, you can render items on your own by setting a label-fn

Controller

$scope.colors2 = [ {
	'name' : 'Red',
	'hex' : '#FF0000'
   }, {
	'name' : 'Orange',
	'hex' : '#FF7F00'
   } ];
   
$scope.myLabelFn = function(item) {
	return item.name + ' (' + item.hex + ')';
}

Markup

<list-group items="colors2" 
	label-fn="myLabelFn(item)"></list-group>
