Built in actions

Use the editable attribute to add an edit action button. 

<list-input-group-item ng-model="item" 
	editable>
</list-input-group-item>

Use the deletable attribute to add a delete action button. 

<list-input-group-item ng-model="item" 
	deletable >
</list-input-group-item>

Use editable and deletable attributes to add an edit and a delete action buttons. 

<list-input-group-item ng-model="item"
	editable
	deletable >
</list-input-group-item>

Use the edit-fn attribute to handle the edit action.

Declare an 'item' named argument to receive the ng-model value. 

<list-input-group-item ng-model="item" 
	editable
	edit-fn="handleEditAction(item)">
</list-input-group-item>

Use the delete-fn attribute to handle the delete action.

Declare an 'item' named argument to receive the ng-model value. 

<list-input-group-item ng-model="item" 
	deletable
	delete-fn="handleDeleteAction(item)">
</list-input-group-item>

Use the inline attribute to enable inline edition of the item 

<list-input-group-item ng-model="oItem.name"
	editable inline>
</list-input-group-item>

model = {{oItem}}

Custom actions

Add custom actions
<list-input-group-item ng-model="item" 
	actions='[
		{"id":"play",
			"icon":"glyphicon-play", 
			"fn":"play(item)"},
		{"id":"pause",
			"icon":"glyphicon-pause", 
			"fn":"pause(item)"},
		{"id":"stop",
			"icon":"glyphicon-stop", 
			"fn":"stop(item)"}]'>
</list-input-group-item>

Button dropdowns actions

Add button dropdowns
<list-input-group-item ng-model="item"
actions='[{
	"icon" : "glyphicon-floppy-disk",
	"actions": [
		{"id":"save",
			"icon":"glyphicon-floppy-save",
			"label" : "Save", 
			"fn":"save(item)"},
		{"id":"remove",
			"icon":"glyphicon-floppy-remove",
			"label" : "Remove",
			"fn":"remove(item)"}	
	]
}]'>
</list-input-group-item>

Sizing

Use the size attribute to set one of the following sizes, large or small .

If no size attribute is specified or with a value other than the ones authorized the item is sized to its default size wich can be considered as medium 

<list-input-group-item ng-model="item" 
	size="large" editable></list-input-group-item>
<list-input-group-item ng-model="item" 
	editable></list-input-group-item>
<list-input-group-item ng-model="item" 
	size="small" editable></list-input-group-item>


Selection

Use the selectable attribute to bind a model property that tracks the selection of the item 

<list-input-group-item ng-model="item"
	selectable="selected"  
	editable>
</list-input-group-item>

Templating

Use the template attribute to customize item content

In your template use item variable name as the reference to ng-model attribute value. 

<list-input-group-item ng-model="edgarDegas" 
	template="template"
	size="large" editable>
</list-input-group-item>

Use the template-url attribute to externalize the content of the template

In your template use item variable name as the reference to ng-model attribute value.

If both template and template-url attributes are specified, only template attribute will be used 

<list-input-group-item ng-model="edgarDegas"
	template-url="'partials/list-input-group-item.tpl.html'"
	size="large" editable>
</list-input-group-item>
